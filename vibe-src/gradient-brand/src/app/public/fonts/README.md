# Fonts Directory

Please place your ServiceNow Sans font files in this directory. Ensure they match the filenames referenced in `/styles/globals.css`:

- `ServiceNowSans-Light.woff2` (Weight: 300)
- `ServiceNowSans-Regular.woff2` (Weight: 400)
- `ServiceNowSans-Book.woff2` (Weight: 450)
- `ServiceNowSans-Medium.woff2` (Weight: 500)
- `ServiceNowSans-Bold.woff2` (Weight: 700)
- `ServiceNowSans-Black.woff2` (Weight: 900)
- `ServiceNowSansMono-Regular.woff2` (Weight: 400 - Monospace)

If you have a variable font file, you can rename it to one of these or update `/styles/globals.css` to point to it.
